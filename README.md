# Flowerbot
# Telegram-бот «Veresk Flowers»

Полноценный интернет-магазин цветов в Telegram с мини‑приложением, синхронизацией с WooCommerce и приёмом платежей через Т‑Банк.

Клиент выбирает букет в каталоге, указывает размер, оформляет заказ и оплачивает его картой или наличными. Администратор получает уведомления, видит статистику и может делать массовые рассылки.

---

## Возможности

-  **Каталог** с категориями, поиском по названию/описанию, сортировкой по цене.
-  **Вариативные товары** — для букетов с разными размерами селектор цены появляется прямо в ленте.
-  **Корзина** и оформление заказа с выбором способа доставки (самовывоз, МКАД, экспресс, за МКАД).
-  **Онлайн‑оплата** картой (интернет‑эквайринг Т‑Банка) или наличными при получении.
-  **Бонусная программа** — 500 приветственных баллов и 5% кешбэк с каждой оплаченной покупки.
-  **Синхронизация заказов** с WooCommerce — заказы автоматически дублируются на сайт.
-  **Рассылка** подписчикам от администраторов.
-  **Статистика** — количество пользователей бота.
-  **Фотогалерея** товаров с возможностью просмотра на весь экран.
-  **Адаптивный дизайн** мини‑приложения под мобильные устройства.

## Установка и настройка

1. Клонируйте репозиторий:
    ```bash
   git clone https://github.com/rzrrz-end/Flowerbot.git
   cd veresk-flowers-bot

2. Установите зависимости:
   pip install -r requirements.txt

3. Заполните файл .env своими данными.
4. Запустите бота:
   python bot.py
  !!Веб сервер стартует на порту 8080, вебхук устанавливается автоматически!!

5. В BotFather укажите адрес мини-приложения:
   /mybots → @YourBot → Bot Settings → Menu Button → Web App
URL: https://....ваше мини приложение.ru

## Команды бота

/start Запуск бота, начисление приветсвтенных бонусов(бонусы начисляются только 1 раз)
/bonus Проверить баланс бонусов
/broadcast Для админов. Разослать сообщение всем пользователям.
/stats Для админов. Посмотреть кол-во пользователей бота.
/отмена Для админов. Отмена рассылки.

## Настройка платежного терминала
1. В Т-Бизнесе создайте боевой терминал (Интернет-Эквайринг)
2. В разделе API Скопируйте Terminalkey и Secretkey
3. Запросите у поддрежки отключение проверки IP-адреса
4. В настройках терминала укажите webhook: https://bot.(url-mini-app)/tbank-webhook
